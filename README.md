## 研究对象、方向与兴趣
我的研究主要围绕地形的气候效应与古气候过程，重点包括：
主要研究对象：风沙、黄土与冰盖
主要研究方向：地表基质与生态功能、风成沉积物物源与古环境演变、第四纪冰盖演化过程及其古气候效应
当前致力于从北方地质记录中解译地表基质的成因与演变机制，揭示其对干旱区生态格局的关键调控作用；同时关注第四纪冰盖演化过程及其古气候效应，以及物源定量化方法的发展与应用。
